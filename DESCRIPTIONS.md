## 🇧🇷 Português (Brasil)

### Descrição Curta (GitHub About)
☁️🛡️ Integre Zscaler ZIA, ZCC e ZDX ao GLPI 11 com dashboard, URL lookup, denylist, Cloud Sandbox, tickets automáticos e cobertura de endpoints.

### Descrição Média
☁️🛡️ **Zscaler para GLPI 11** conecta segurança web, inventário e service desk. Sincronize categorias e denylist do ZIA, consulte URLs, use Cloud Sandbox, monitore endpoints sem ZCC, acompanhe alertas ZDX e gere tickets automáticos com trilha de auditoria.

### Descrição Longa
🚀 **Zscaler GLPI Plugin** transforma o GLPI 11 em um ponto operacional para Zscaler, unindo ZIA, ZCC e ZDX em dashboards, listas, ações controladas e tickets rastreáveis.

✨ Funcionalidades:
- 🧭 Dashboard com categorias, denylist, ações recentes e cobertura
- 🌐 Integração com ZIA para categorias, denylist e URL lookup
- 🧪 Cloud Sandbox com consulta por hash e submissão de arquivos
- 🚫 Bloqueio/remoção de URLs com trava dupla de segurança
- 🧩 Recategorização em categorias customizadas
- 💻 Inventário ZCC e relatório de computadores sem Zscaler
- 📈 Alertas ZDX sincronizados com abertura de tickets por severidade
- 🎫 Tickets automáticos para ações, sandbox malicioso e alertas
- 🔐 Permissões dedicadas por perfil GLPI
- 🧾 Logs de auditoria para ações e sincronizações

🎯 Ideal para equipes de TI, segurança, SOC e service desk que precisam investigar URLs, aplicar bloqueios com controle, medir cobertura de endpoints e registrar incidentes no GLPI.

## 🇺🇸 English

### Short Description (GitHub About)
☁️🛡️ Integrate Zscaler ZIA, ZCC, and ZDX with GLPI 11 using dashboards, URL lookup, denylist actions, Cloud Sandbox, automatic tickets, and endpoint coverage.

### Medium Description
☁️🛡️ **Zscaler for GLPI 11** connects web security, inventory, and service desk workflows. Sync ZIA categories and denylist data, run URL lookups, use Cloud Sandbox, monitor endpoints without ZCC, track ZDX alerts, and generate automatic GLPI tickets with audit history.

### Long Description
🚀 **Zscaler GLPI Plugin** turns GLPI 11 into an operational hub for Zscaler by combining ZIA, ZCC, and ZDX into dashboards, searchable lists, controlled actions, and traceable tickets.

✨ Features:
- 🧭 Dashboard with categories, denylist, recent actions, and coverage
- 🌐 ZIA integration for categories, denylist, and URL lookup
- 🧪 Cloud Sandbox hash lookup and file submission
- 🚫 Controlled URL block/remove actions with safety locks
- 🧩 Custom category recategorization
- 💻 ZCC inventory and computers without Zscaler report
- 📈 ZDX alerts synced with severity-based ticket creation
- 🎫 Automatic tickets for actions, malicious sandbox results, and alerts
- 🔐 Dedicated GLPI profile permissions
- 🧾 Audit logs for actions and synchronizations

🎯 Built for IT, security, SOC, and service desk teams that need URL investigation, controlled blocking, endpoint coverage visibility, and incident tracking inside GLPI.
